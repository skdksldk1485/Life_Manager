#### 사용기술
- HTML5 / CSS3
- Typescript
- React
- GraphQL
- next.js
- Apollo
- Styled-components, material-ui

#### 구현기능
- 회원가입/로그인/로그아웃
- 동영상 stream CRUD 
- 랜딩페이지 scrollDown, scrollUp
- 반응형 웹
